We stopped updating property boundary lines and city annexation information on section maps since April 25th, 2012. County Road Number Search Examples:

  • Please add leading zeros in the following cases: using 0001 to search county road number 1; using 0010 to search county road 10, etc.
  • To search county road with a fractional number, using 1338-5 for searching 1338 1/2; A190-75 for A-190 3/4, etc
  • To search County road with "A", using 0A24 for searching A-24; 00A1-25 for A-1 1/4; A177 for A-177, etc.
  • To search county market roads, using mr21 for searching market road 21.
